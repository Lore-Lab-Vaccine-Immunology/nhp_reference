Calculating ratios

This section calculates four important blood cell ratios used in hematology: 1. NLR (Neutrophil-to-Lymphocyte Ratio) - inflammation marker 2. MLR (Monocyte-to-Lymphocyte Ratio) - immune system indicator 3. PLR (Platelet-to-Lymphocyte Ratio) - inflammation and thrombosis marker 4. NMR (Neutrophil-to-Monocyte Ratio) - immune response indicator

The calculated ratios are added as new columns to the dataset.

clean_dataset <- clean_dataset %>%
  mutate(NLR = (`measurement_value_Neutrophil-Band-Abs`+`measurement_value_Neutrophil-Seg-Abs`)/ `measurement_value_Lymphocytes-Abs`,
         MLR = `measurement_value_Monocytes-Abs`/`measurement_value_Lymphocytes-Abs`,
         PLR = measurement_value_Platelets/`measurement_value_Lymphocytes-Abs`,
         NMR = (`measurement_value_Neutrophil-Band-Abs`+`measurement_value_Neutrophil-Seg-Abs`)/`measurement_value_Monocytes-Abs`
         )

write.csv(clean_dataset, file = paste0("../", result.dir, "/merged_dataset_ratios.csv"), 
          row.names = F)

Simple visualization

This section creates violin plots to visualize the distribution of blood cell ratios: 1. First transforms the data from wide to long format for plotting 2. Creates age groups (Juvenile, Young Adults, Adults, Middle-Aged, Elderly) 3. Generates separate plots for each age group using facet_wrap 4. Uses log10 scale for better visualization of ratio distributions 5. Colors plots by species for comparison 6. Saves the final plot as a TIFF image in the figures directory

clean_dataset %>%
  pivot_longer(cols = c(NLR, MLR, PLR, NMR), values_to = "ratio_value", names_to = "ratio") %>%
  mutate(age_group = factor(case_when(age <= 3 ~ "Juvenile",
                               age > 3 & age <= 7 ~ "Young Adults",
                               age > 7 & age <= 12 ~ "Adults",
                               age > 12 & age <= 17 ~ "Middle-Aged",
                               age > 17 ~ "Elderly"
                               ), levels = c("Juvenile", "Young Adults", "Adults", "Middle-Aged", "Elderly") )) %>%
  ggplot(aes(x = ratio, y = ratio_value, color = species_common_name)) +
  geom_violin(trim = FALSE, scale = "area", draw_quantiles = TRUE) +
  scale_y_log10() + 
  labs( y = "Ratio values (log10)", x = "", color = "") +
  facet_wrap(~age_group) +
  theme_classic() +
  theme(aspect.ratio = 1)
